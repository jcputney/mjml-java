# Code of Conduct

## Our standard

This project follows a professional, respectful collaboration standard:

- Be respectful and constructive.
- Focus feedback on ideas and code, not people.
- Assume good intent.
- Avoid harassment, discrimination, and personal attacks.

## Expected behavior

- Provide actionable technical feedback.
- Be open to different viewpoints.
- Help maintain a welcoming environment for contributors of all backgrounds.

## Unacceptable behavior

- Harassment, intimidation, or hate speech.
- Personal insults, trolling, or bad-faith arguments.
- Publishing private information without consent.

## Enforcement

Project maintainers may edit/remove content, lock threads, or restrict participation for behavior that violates this code.

## Reporting

If you experience or witness unacceptable behavior, open a private security discussion via `SECURITY.md` contact guidance.
