Line Heights
Line-height 1.0 - This is very tight spacing between lines. Multiple lines of text will appear cramped and hard to read. Not recommended for body text in emails.
Line-height 1.4 - This is a moderate spacing that works well for most body text. It provides enough breathing room while keeping content compact and efficient.
Line-height 1.8 - This is generous spacing between lines. It makes longer paragraphs much easier to read but uses more vertical space. Good for important messages or luxury brand emails.
Line-height 24px - Using pixel values for line-height provides exact control over the spacing between lines, regardless of the font size.
Letter Spacing
Letter-spacing: 0px (normal) - Default character spacing.
Letter-spacing: 1px - Slightly expanded, gives a clean look.
Letter-spacing: 2px - More open spacing, good for headings.
Letter-spacing: 4px uppercase - Dramatic spacing for labels
Text Decoration & Transform
Text-decoration: underline - Adds an underline to all text in this block.
Text-decoration: overline - Adds a line above the text.
Text-decoration: line-through - Strikes through the text for deletions.
Text-transform: uppercase - converts all characters to uppercase.
Text-Transform: Lowercase - CONVERTS ALL CHARACTERS TO LOWERCASE.
text-transform: capitalize - capitalizes the first letter of each word.
