Navbar Without Hamburger
The navbar above renders links inline without a hamburger toggle. On mobile devices, the links will stack vertically. This mode is best when you have a small number of navigation items.

Navbar With Hamburger Menu
The red navbar above uses hamburger="hamburger" mode. On supported email clients, a hamburger icon (☰) appears on mobile, and clicking it reveals the navigation links. The close icon (✕) dismisses the menu. This mode is ideal for navbars with many links.

Custom-Styled Navbar Links
The teal navbar above uses custom font-family (Georgia), normal font-weight, no text-transform, and one link has an underline decoration. This demonstrates that each mj-navbar-link can be individually styled.
