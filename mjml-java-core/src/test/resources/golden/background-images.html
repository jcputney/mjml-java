Background Size: Cover
The background-size is set to "cover", which scales the image to fill the entire section while maintaining its aspect ratio. Parts of the image may be cropped if the aspect ratios do not match.
Background Size: Contain
With background-size set to "contain", the image is scaled to fit entirely within the section without cropping. This may leave gaps on the sides or top/bottom.
Background Size: 200px 200px
This background uses explicit pixel dimensions for the background-size property, creating a tiling pattern of 200x200 pixel images.
Background Position: Top Center
The background-position is set to "top center", which anchors the background image to the top-center of the section. This is useful when the important part of the image is at the top.
