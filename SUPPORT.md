# Support

## Getting help

Use one of the following:

- Documentation: https://jcputney.github.io/mjml-java/
- GitHub Issues: bug reports and feature requests
- GitHub Discussions (if enabled): usage questions

## Before opening an issue

- Confirm you are on a recent version.
- Check existing issues for duplicates.
- Prepare a minimal MJML reproduction.
- Include Java version, OS, and module(s) used.

## Scope

This repository supports:
- `mjml-java-core`
- `mjml-java-resolvers`
- `mjml-java-spring`
- `mjml-java-bom`
